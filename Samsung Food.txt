- [Server Token](https://docs.whisk.com/api-overview/auth/server-token): A server token allows you to enable server-to-server integration with Whisk for non-personalized data.
- [User Access Token](https://docs.whisk.com/api-overview/auth/user-access-token): A user access token authorizes you to read, modify or write data on behalf of a specific Whisk user.

- [Client Token](https://docs.whisk.com/api-overview/auth/client-token): A client token allows you to access a limited subset of Whisk APIs in read-only mode.
